# HarenKei Next.js Boilerplate

> 본 프로젝트는 개인용 Next.js 보일러플레이트를 만드는 프로젝트입니다.
> Next.js 프로젝트에서 개인적으로 자주 사용하는 초기 설정을 미리 해두었습니다.
> Github Template Repository로 설정하여 해당 초기 설정에서 분기하여 바로 개발을 시작할 수 있도록 하였습니다.
